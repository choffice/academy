화살표 함수

  1. arrow function :: 함수의 축약형
  2. 원형함수는 불가, 익명함수만 변형 가능
  3. *비교연산자는 >=로, 반드시 등호가 뒤에 가야한다
  4. 명령문이 하나일 경우 {}는 생략이 가능하다
  5. 매개변수가 하나일 경우 ()도 생략이 가능하다
  6. 익명함수 방식?! console.log(var anonyfnc = function (){console.log('익명함수 방식');}anonyfnc(););!!//function(){}
  7. 화살표 방식?! console.log(var arrowfnc = () => {console.log('화살표 방식');}arrowfnc(););!!//()=>{}, (a,b)=>{}, a=>{}, a=>;
  8. 화살표 함수의 특징 :: 다른 명령문 없이 return만 있을 때 {return값}만 남기고 return명령어는 생략이 가능하다
