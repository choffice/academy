자바스크립트 변수의 타입 typeof

타입과 형변환

변수의 타입

  • 원시타입(Primitive Type) : 값 자체가 메모리에 직접 저장
  • number : 숫자(정수, 실수)
  • bigint : 숫자인데 매우 큰 수(n이 붙음), 크기가 아니라 표현방식
  • string : 문자열
  • boolean: true, false
  • undefiend : 값이 초기화되지 않을 때 값을 출력
  • null : 값이 없음을 의도적으로 명시할 때 사용. *type은 object
  • symbol : 절대 중복되지 않는 고유한 식별자를 만들 때 사용(ES6), 정의할 땐 Symbol(), type은 symbol
  • 참조타입(Referenced Type) : 주소값, 주소에 가면 집합군을 접근
  • - 1)함수의 이름 그대로 타입을 보는 것과, 2)함수를 작동()시키고 타입을 보는 것은
    각각 1) function 타입, 2) object 또는 return 값의 타입

    - 변수에 담을 때도
    function()으로 담으면 변수 자체가 (클래스의) 함수명이 되고,
    함수명()으로 담으면 리턴값이 담기는 것
  • object
  • Array(object)
  • function
  • DOM
  • 내장 객체(Date, Math, ...)

변수의 형 변환

  • var tot = 100으로 초기화했을 때, ... type은 number
  • tot = true로 할당했을 때, ... type은 boolean
  • var와 let은 타입 변환이 자유롭다(자바스크립트 특징)
