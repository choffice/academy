변수 variable 선언과 종류


선언과 명명법, 종류


  • 변수의 이름은 영문자(대소문자)와 숫자, _, $만 사용
  • 변수의 시작은 숫자를 제외한 나머지로 시작
  • 변수의 이름에 두 단어 이상이 합쳐지면 Camel Expression 적용
  • 변수명은 다른 사람이 보더라도 변수의 의미를 이해할 수 있는 이름을 사용
  • 예약어(var, let, null, undefined, number, for, if ...) 제외

변수의 선언

  • <변수의 구조> 제어자, 변수명=식별자

  • num : 초기화되지 않은 상태 (사용X)
  • num = 10 : (선언) + 초기화, 암묵적 전역 변수로 생성되어 충돌 위험
  • var num = 10 : 선언(제어자) + 초기화, 구버전
  • let num = 10 : 선언(제어자) + 초기화, 신버전
  • const num = 10

- 초기화 : 맨 처음의 할당

호이스팅과 참조에러

  • hoisting

    자바스크립트는 실행 전에 변수 선언을 먼저 처리하는데 마치 코드 상단에 선언한 것처럼 사용한다. 그래서 변수 선언만 상단으로 가져가고 초기화는 해당 라인에서 진행하는 형태

  • *특징 : 선언은 상단으로, 초기값은 undefined, 에러는 안 나게
  • var : 선언 전에 사용 가능, 값은 초기화 되기 전 undefined
  • let : 선언 전에 사용 불가, 초기화 전 값은 undefined
  • const : 선언+초기화 전에 사용 불가, 최초 할당이 마지막 할당이므로
  • *참조(referenced)할 수 없는 TDZ(일시적 사각지대)가 발생
