CSS 함수

  • 결정함수 : is(), not()
    • 같은 클래스를 모두 넣어줬지만 h2는 is함수의 적용을 받고, div는 not으로 부정당해서 적용되는 스타일이 없으므로,
      is(h2)이면서 not(div)인 h2만 스타일을 적용
      => is()와 not()은 css에서 클래스를 정의하는 함수
    div

    h2

  • 변수함수 : var()
    • --로 변수값을 정의하고 var()안에 변수명을 넣어 값을 대체어로 재사용하는 함수

    not

  • 속성함수 : attr()
    • 해당 선택자에 속성을 적어주는 함수(속성을 노드텍스트처럼)
    홈으로
  • url 함수 : url()
    • 외부의 소스를 당겨오는 함수
    my url
  • 색상 함수 : rgb(), rgba(), hsl(), hsla()

    • my color1

    my color2

  • 계산 함수 : calc()
    • *띄어쓰기 주의(값_단위 - 값_단위)
    계산할 수 있다.
  • 최대 최소 함수 : max(), min()
    • 서로 다른 단위 간에 비교하여 더 큰/작은값을 반영 (반응형으로 스크린 비율이 달라질 때 유용)
    두개의 인자 중 큰,작은 값을 반환하는 css함수
  • clamp 함수 : clamp()
    • 순서에 따라 스펙을 잘 정의해야 함, 사이즈 간의 우선순위
    최소 크기, 이상적 크기, 최대 크기를 반환하는 함수
  • 모양 관련 함수 : circle(), polygon()

    • clip-path로 모양을 사용하여 상자를 시각적으로 다르게 보여지는 함수
