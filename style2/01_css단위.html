%, em, cm, mm, inch, px, pt, ex, ch 등

일반적인 16px = 12pt = 1em = 100%

  1. 절대적인 크기 pt, px, cm, mm, in, pc, Q
  2.  
    1. pt :: 1pt = 1 in의 1/72
    2. px :: 1px = 1 in의 1/96
    3. cm :: 1cm = 1 in의 96/2.54
    4. mm :: 1mm = 1 cm의 1/10
    5. in :: 1in = 2.54 cm = 96px
    6. pc :: 1pc = 1 in의 1/16
    7. Q :: 1Q = 1 cm 1/40
  3. --
  4. 상대적인 크기 em, %

    5. rem(root em) :: px, pt가 절대적인 크기라면 em, %는 상대적인 크기
    - 부모 요소의 크기에 따라 유연하게 변하는 반응형의 단위

    font size( 14px )
    font size (14 * 1.2 = 16.8px)
    font size(16.8 * 1.2 = 20.16px)
    font size(20.16 * 1.2 = 24.192PX)
  5. 각도 단위 deg, grad, rad, turn  
    • deg :: 도를 지칭, 360도가 완전한 원
    • grad :: 그라디안을 지칭, 400 그라디안이 완전한 원
    • rad :: 라디안을 지칭, 2π 라디안이 완전한 원
    • turn :: 턴을 지칭, 1턴이 완전한 원
  6. 시간 단위 s, ms  
    • s :: 초
    • ms :: 밀리초, 1000ms = 1s
  7. 해상도 단위 dpi, dpcm, dppx, x  
    • dpi :: 인치당 점의 수.
    • dpcm :: 센티미터당 점의 수.
    • dppx, x :: 픽셀당 점의 수.
  8. 모바일 단위 vw, vh  
    • vw :: 뷰포트 너비의 1%.
    • vh :: 뷰포트 높이의 1%.
